Platform Summary

This platform delivers remote sensing–based environmental intelligence across active mining zones in the Pilbara region. Multi-temporal Sentinel-2 imagery is processed for NDVI change (vegetation loss) and NDBI change (bare soil expansion), with automated hotspot extraction producing a ranked inventory of detected disturbance sites to support inspection planning, ESG monitoring, and compliance workflows.

About

Platform

This dashboard is a remote sensing–based environmental monitoring prototype for detecting mining-related land disturbance. It uses satellite-derived indices and WebGIS visualisation to support preliminary remote sensing screening, inspection planning, ESG monitoring, and environmental compliance support workflows.

Methodology

Environmental change detection uses multi-temporal NDVI differencing on Sentinel-2 MSI imagery to quantify vegetation loss, combined with NDBI change analysis to capture bare soil and built-up surface expansion. Disturbance hotspot polygons were delineated using object-based image analysis (OBIA) and classified into severity tiers by change magnitude.

Technical Notes

Polygon area values displayed in the Hotspots and Analytics panels are estimated client-side from geographic bounding-box geometry using a 0.65 fill-factor correction. These are proxy values suitable for relative ranking and screening triage — not survey-grade measurements. Authoritative areas require analysis in a projected coordinate system.
